## Generator Flavor Text

### Entropy Era

#### Random Number Generator
```
"Pure chaos. Maximum entropy. One coin flip at a time."

The most fundamental source of information - complete
unpredictability. Each bit is truly random, containing
exactly 1 bit of information.
```

#### Biased Coin
```
"Not all outcomes are equal. Less surprise, more data."

When the coin lands on heads 70% of the time, each flip
contains ~0.88 bits of information. Predictability reduces
entropy, but data generation increases.
```

#### Dice Roller
```
"Six possible states. log‚ÇÇ(6) ‚âà 2.58 bits per roll."

A fair six-sided die produces 2.58 bits of information per
roll. More states, more information - but still discrete,
still random.
```

#### Deck Shuffler
```
"52! permutations. Astronomical entropy."

A shuffled deck has 52 factorial possible arrangements.
That's about 226 bits of information in a single shuffle.
Combinatorial explosion at work.
```

#### Noise Generator
```
"White noise. Every frequency, equal power."

Thermal noise from quantum fluctuations. The universe's
own random number generator, producing truly random bits
from fundamental physics.
```

#### Thermal Sensor
```
"Brownian motion. The universe's random number generator."

Measuring the random jitter of molecules in motion. Maximum
entropy from minimum predictability - nature's chaos engine.
```

### Compression Era

#### Run-Length Encoder
```
"AAAAAAA becomes 7A. Simple patterns, simple gains."

The simplest compression algorithm. Replaces repeated
sequences with count + symbol. Effective on low-entropy
data with long runs.
```

#### Huffman Tree
```
"Common symbols = short codes. Optimal prefix-free encoding."

Builds a binary tree where frequent symbols get short codes.
Provably optimal for symbol-by-symbol encoding. Created by
David Huffman in 1952.
```

#### LZ77 Dictionary
```
"Already seen this sequence. Backreference: (distance, length)."

Invented by Lempel and Ziv in 1977. Replaces repeated
sequences with pointers to earlier occurrences. The foundation
of ZIP, GZIP, and PNG compression.
```

#### Arithmetic Coder
```
"Encode entire message as a single number in [0,1)."

Maps messages to intervals on the number line. Asymptotically
optimal - approaches the Shannon entropy limit. Used in JPEG
and modern video codecs.
```

#### Context Mixer
```
"Predict next bit from weighted context history."

Maintains multiple context models and adaptively weighs their
predictions. State-of-the-art compression for text and certain
data patterns.
```

#### Neural Compressor
```
"Learn the data manifold. Reality has structure."

A neural network learns the underlying structure of data,
then compresses based on learned patterns. The frontier of
compression research.
```
